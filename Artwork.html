What is classified as Game Art?

Definition: Game Art is any visual elements players see when playing a game

When creating artwork for a game it depends on what style the game is aiming for or if the game is 2D or 3D. Both of these will drastically change how long time is spent on it.

These images are from Valorant as it was being developed, which took 6 years of development before it launched

Industry Software:

Blender - This is a 3D modeling software and it is used to make a variety of 3D models from guns to characters

Photoshop - This is graphics editing software and it is used to create a variety of artwrok from user interfaces to concept art

Job Roles & Education Requirements

  • 3D Asset Artist
  • 3D Environment Artist
  • Animation Artist
  • 2D Illustrator

All of these jobs require a Bachelor's Degree in Graphic Design, Animation, Fine Arts, or Similar
